# wec_2026

Repository for the Warsaw Econometric Challenge 2026.

## Files

*merge.ipynb*: contains the pipeline to create *very_final_dataset.csv*.

*shap.ipynb*: produces the SHAP analysis of the covariates.

*eda.ipynb*: contains the exploratory factorial analysis.

*all_models.ipynb*: is the code used for the full analyses.

*paper.pdf*: contains the main findings.

*very_final_dataset.csv*: augmented dataset used for the analysis.

*players_quarters_final.csv*: original dataset.

*player_appearance_run.csv*: dataset with sprinting data.

*player_appearance_pass.csv*: dataset with passing data.

*player_appearance_behaviour_under_pressure.csv*: dataset with behavipur under pressure.
